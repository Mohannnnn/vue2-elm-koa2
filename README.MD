### vue2-elm-koa2后端系统
> 前端github地址:[https://github.com/Mohannnnn/vue2-elm](https://github.com/Mohannnnn/vue2-elm)

### 技术栈
`koa2` + `node.js`

### 开始
```bash
npm install

npm install nodemon -g  //本地开发监听文件变化
npm run dev             //本地开发测试

npm run server          //上线开启服务
```
### 目录结构
```
|-dist                      //打包上传的前端静态文件路径
|-server
|       controller
|       midddleware
|       routers
|       utils
|-config.js                 //配置文件       
|-index.js                  //koa服务入口文件
|-README.MD
```
